Need Help?

Make sure:

  • The RSCM IP address is correct and accessible from this machine
  • Redfish API is enabled on the RSCM (usually enabled by default)
  • You have valid RSCM credentials (typically root/password)
  • Network connectivity allows HTTPS on port 8080 to the RSCM
